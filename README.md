# Description
It is a list of tasks you need to complete or things you want to do where you can add your tasks very easily.

# How to run the project
```bash
# Copy repository
$ git clone 
# Install dependecies
$ npm install
or
$ yarn install
# Run application
$ npm start
ou
$ yarn start
```

Access: http://localhost:3000/
